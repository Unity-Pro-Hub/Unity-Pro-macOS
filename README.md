## Why Unity Pro LTS is the choice of pros

- Native Apple Silicon support (M1–M4)  
- Splines, Multiplayer tools, Sentis (AI on device)  
- Enhanced Netcode for GameObjects  
- Improved HDRP ray tracing & volumetric fog  
- Faster build times + cloud diagnostics  

This community toolkit gives you **ready-to-use pro assets and setups** that save weeks of work.

- `Project Templates/` — 2D Platformer, 3D FPS, Mobile RPG, VR Starter

## Installation (macOS) – 60 seconds

1. Click the button → archive downloads  
2. Drag folders into your Unity project’s Assets  
3. Restart Unity → everything is ready to use
